About this App

Track your MP

The Track your MP app helps you identify the MP of an area and keep track of their activty in the House of Commons, in the politics section of the news and on Twitter.

Track your MP was created by Tuck Software, using the following software:

  • Icenium, based on Cordova
  • JQuery and JQuery Mobile

And the following data sources:

  • They Work for You API
  • Guardian Open Platform: Content API
  • Twitter API
  • Google Map API

Should you find any errors or discover innaccurate information, please email us at tucksoftware@gmail.com so that we can fix it.
